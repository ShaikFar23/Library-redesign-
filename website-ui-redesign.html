Part 01 — Heuristic Evaluation
Usability Issues Found in Existing Site
Website Evaluated
City Public Library (existing site)
Evaluation Method
Nielsen's 10 Usability Heuristics
Issues Found
9 Issues (3 High · 4 Medium · 2 Low)
Evaluators
UX Design Team · 2025
● High Severity

No Clear Visual Hierarchy

All text appears in the same size and weight. Users cannot distinguish headings from body text, making it impossible to scan content quickly.

Fix: Establish a clear type scale with distinct heading sizes
● High Severity

Search is Buried & Unclear

The book search function is hidden on a secondary page. Users expect search to be the primary action but cannot find it on the homepage.

Fix: Place a prominent search bar as the hero element
● High Severity

Not Mobile Responsive

The website breaks on screens smaller than 1024px. Text overflows, buttons become unclickable, and images get cropped incorrectly.

Fix: Rebuild using a mobile-first responsive layout
● Medium Severity

Poor Colour Contrast

Light grey text on white background fails WCAG AA contrast requirements. Users with visual impairments cannot read key content.

Fix: Ensure minimum 4.5:1 contrast ratio for all text
● Medium Severity

Inconsistent Navigation

The navigation menu changes structure between pages. Some pages show 6 items, others show 9, with different ordering each time.

Fix: Create a consistent global navigation component
● Medium Severity

No Feedback on Actions

When users reserve a book, no confirmation message appears. They are left unsure whether their action was successful.

Fix: Add success/error states and confirmation messages
● Medium Severity

Cluttered Homepage

The homepage contains 14 different sections with no clear priority. Users experience decision fatigue and leave without completing tasks.

Fix: Reduce to 4–5 focused sections with clear purpose
● Low Severity

Outdated Typography

The site uses Times New Roman throughout, which feels dated and reduces perceived credibility of the institution.

Fix: Adopt a modern, readable typeface system
● Low Severity

Missing Empty States

When a book search returns no results, a blank white page appears with no guidance on what to do next.

Fix: Design helpful empty states with suggested actions
